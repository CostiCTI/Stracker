What's STRACKER

STRACKER is a program that aims to increase the quality of software development by tracking and suggesting the dimensions of software metrics.



Create a project

To create a new project you must go to the Projects. At the bottom of the page, in the text box you must enter the name of the new project and then click the Create button. A new project has been created.



Open a project

If you want to open an existing project, click on the blue button with the name of the project you want. If you want to delete a project, click the x button for the project you want to delete



Work with a project

In a project you can track and analyze the software metrics of a project. You can select the metric chart you want to view by selecting the appropriate button. You can add a new stage of the project and you can check if the last update was better than the previous one.
